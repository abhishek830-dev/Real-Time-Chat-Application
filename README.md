# Real-Time Chat Application

A simple and lightweight **Real-Time Chat Application** built using **Node.js**, **Express.js**, and **Socket.IO**.  
It allows multiple users to connect and exchange messages instantly in real time.

---

## 🚀 Features

- Real-time messaging using Socket.IO  
- Multiple users can join and chat  
- Instant message broadcasting  
- Clean and simple UI  
- Runs locally on your system  

---

## 🛠 Tech Stack

- **Node.js** – Backend runtime  
- **Express.js** – Web framework  
- **Socket.IO** – Real-time communication  
- **HTML, CSS, JavaScript** – Frontend
